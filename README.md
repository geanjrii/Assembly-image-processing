# Programa de Modificação de Imagens BMP

Este programa permite a modificação de imagens no formato BMP. Ele solicita ao usuário o nome do arquivo de entrada e do arquivo de saída, bem como a seleção de uma cor e o valor de aumento dessa cor. O programa abrirá o arquivo de entrada, lerá o cabeçalho do arquivo BMP e, em seguida, lerá e modificará cada pixel da imagem de acordo com a cor selecionada e o valor de aumento especificado. O resultado será salvo no arquivo de saída.
